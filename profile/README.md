# The Set-Top Review

A device-first publication. We review the hardware that streams the
content, not the content. If the box matters to your living room, we cover it.

## What's in scope

- Streaming sticks (Fire TV Stick 4K, Fire TV Stick 4K Max, Roku Streaming
  Stick 4K, Chromecast with Google TV)
- Streaming boxes (Apple TV 4K, NVIDIA Shield TV, Roku Ultra)
- Smart TVs as streaming platforms (Tizen, webOS, Google TV, Roku TV)
- IPTV apps that run on those devices — but always reviewed in the context
  of the device, not standalone

## What's out of scope

- Phones / tablets as primary streaming devices
- Browser-based streaming
- Console-based streaming (covered elsewhere)

## How we review

- We buy retail. No review units sent by manufacturers stay in our test pool
  past the first week.
- Every review includes the actual hardware revision and firmware version
  tested, because IPTV app behaviour changes between firmwares.
- We retest annually. Last year's verdict is left visible with a "superseded"
  note — never deleted.

Adjacent reading: technical IPTV-service testing at https://streamreviewhq.com,
provider-side install walkthroughs at https://iptvtheone.com.
